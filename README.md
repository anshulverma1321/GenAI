# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/anshulverma1321/GenAI.git
```

Move into the project

```bash
cd GenAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Running Examples

Example:

```bash
python chat.py
```

Run Gemini chatbot

```bash
python chat_gemini.py
```

Run Weather Agent

```bash
python weather_agent.py
```

Run RAG example

